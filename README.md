## 001 - JS打气球例子
参考b站上的视频例子拷贝/学习的代码。
代码实现的功能

 - 利用Javascript生成一定数量的气球（气球通过CSS画出来）
 - 气球以不同的速度网上升
 - 点击气球，气球会“爆炸”（跑远+变小）
